# 🚌 LiveBus GIETU – Real-Time Bus Tracking System

## 📱 Overview

LiveBus GIETU is an Android application developed to provide **real-time tracking of university buses** for students and staff of GIET University. The app helps users monitor bus locations, estimate arrival times, and improve daily commute planning.

This project focuses on **real-time data processing, location tracking, and user-friendly UI**, making transportation more efficient and reliable.

---

## ✨ Features

* 📍 Real-time Bus Location Tracking
* ⏱️ Estimated Arrival Time (ETA)
* 🗺️ Google Maps Integration
* 📡 Live Data Sync using Firebase
* 📱 Clean and Responsive UI
* 🔔 Easy Navigation and User Interaction

---

## 🛠️ Tech Stack

* **Language:** Java / Kotlin
* **IDE:** Android Studio
* **Maps API:** Google Maps SDK
* **Backend:** Firebase Realtime Database
* **Location Services:** GPS / Location API
* **Version Control:** Git & GitHub

## 📥 Installation

```bash
git clone https://github.com/Fpvsourav955/LivebusGIETU.git
```

### Steps:

1. Open project in Android Studio
2. Sync Gradle
3. Add your Google Maps API key
4. Connect Firebase
5. Run the app on emulator/device
